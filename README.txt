CMCS ASP.NET Core MVC prototype (GUI only, no functionality)

How to run:
1. Ensure .NET 7 SDK is installed.
2. In this project's folder run:
   dotnet restore
   dotnet run
3. Open https://localhost:5001 or the URL shown in the console.

Notes:
- This is a GUI-only prototype. Buttons and forms are non-functional.
- Data shown in the views is static placeholder data.
